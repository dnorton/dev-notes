<document># [Zed](https://zed.dev/)

Super fast editor written in rust

## Useful keybindings (https://zed.dev/docs/key-bindings)

| Shortcut | Action |
| --- | --- |
| `⌘ + ?` | Toggle Focus (remember this needs a `Shift`) |
| `⌘ + J` | Toggle bottom dock |
| `⌘ + B` | Toggle left dock |
| `⌘ + R` | Toggle right dock |
| `Control + ~` | New terminal |
